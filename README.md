# 감사보고서 검색 시스템 - 데이터 업로드 앱

감사원 감사보고서 데이터를 Gemini File Search Store에 업로드하는 Windows 프로그램입니다.

## 📦 다운로드 및 설치

1. 우측 Release에 있는 `upload_app_v0.0.0.zip` 파일을 다운로드합니다.
2. 압축을 풀고 `upload_app.exe`를 실행합니다.
3. Google AI Studio API 키를 입력하고 업로드를 시작합니다.

## 📋 포함 내용

- `upload_app.exe` - 업로드 프로그램 (Windows용)
- `README.md` - 사용 설명서

## 🔑 API 키 발급

[Google AI Studio](https://aistudio.google.com/apikey)에서 무료로 API 키를 발급받을 수 있습니다.

## ⚠️ 주의사항

- Windows 전용입니다.
- 업로드에는 긴 시간이 소요될 수 있습니다 (총 10,500여 개 파일, 분당 10여 개 업로드).
- 중단되면 다시 실행하세요. 이어서 진행됩니다.
